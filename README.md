Текстовая игра про водителя, который едет через границу на работу. Проект сделан на C++ с использованием библиотеки SFML.
Игра рассказывает историю молодого водителя, который учится у наставника и потом отправляется в путь через границу. По ходу игры принимаются решения, которые влияют на развитие персонажа и его навыки.
Основная механика - это система блокнота, где показываются воспоминания и текущие события. Игрок выбирает варианты действий, получает опыт и артефакты.

Как запустить
системные требования:

macOS или Linux
CMake версии 3.16 или выше
Компилятор C++17
SFML 3.


сборка:

mkdir build
cd build
cmake ..
make

запуск:

cd build
./kkurs
