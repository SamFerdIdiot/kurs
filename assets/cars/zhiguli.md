# Жигули (Zhiguli) - Asset Requirements / Требования к ассетам

## Описание / Description
Народный автомобиль. ВАЗ-2106 "Жигули" - самый массовый советский автомобиль. Компактный, экономичный, узнаваемый. Машина, на которой ездила вся страна.

## Технические характеристики / Technical Specifications
- **Тип**: Седан малого класса / Compact Sedan
- **Объем бака**: 43 литра
- **Расход топлива**: 12 км/л (самый экономичный)
- **Грузоподъемность**: 400 кг
- **Скорость**: Выше средней (140 км/ч)
- **Прочность**: Средняя (1.0x)

## Требования к спрайтам / Sprite Requirements

### Вид сбоку (Side View)
- **Размер**: 100x70 пикселей (меньше чем Волга)
- **Формат**: PNG с прозрачностью
- **Цвета**: Белый, синий, красный - классические цвета "шестерки"
- **Детали**: Характерная угловатая форма, квадратные фары
- **Состояния**: 
  - Идеальное состояние (100%)
  - Потрепанное (50%)
  - Разбитое (0%)

### Вид сверху (Top-Down View)
- **Размер**: 56x112 пикселей
- **Формат**: PNG с прозрачностью
- **Угол**: Строго сверху для игрового вида
- **Те же состояния** что и для бокового вида

### Иконка в меню (Menu Icon)
- **Размер**: 48x48 пикселей
- **Стиль**: Упрощенный силуэт "шестерки"
- **Формат**: PNG с прозрачностью
- **Цвет**: Монохромный или 2-3 цвета

## Исторический контекст / Historical Context
ВАЗ-2106 "Жигули" (в народе "шестерка") производилась с 1976 по 2006 год. Самая популярная модель Волжского автозавода. В 90-е была везде - от такси до семейных машин. Ремонтопригодная, запчасти продавались на каждом углу.

## Атмосфера для арта / Art Direction
- **Эпоха**: 1980-90е годы
- **Состояние**: Разная степень износа, часто видавшая виды
- **Детали эпохи**: Может быть тюнинг 90-х (спойлер, тонировка, наклейки)
- **Цветовая гамма**: Яркие заводские цвета, но с патиной времени

## Примечания для художника / Artist Notes
- Сохранить характерную угловатую форму ВАЗ-2106
- Квадратные фары - важная деталь
- Компактнее чем "Волга", но узнаваемая
- Для пиксель-арта: подчеркнуть квадратные формы
- Можно добавить детали тюнинга 90-х (по желанию)

## Особенности / Special Features
- Самая распространенная машина 90-х
- Узнаваемый силуэт
- Возможны вариации тюнинга эпохи
