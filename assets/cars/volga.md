# Волга (Volga) - Asset Requirements / Требования к ассетам

## Описание / Description
Надежная советская легенда. ГАЗ-24 "Волга" - символ советского автопрома 1970-80х годов. Просторный, надежный седан для дальних поездок.

## Технические характеристики / Technical Specifications
- **Тип**: Седан / Sedan
- **Объем бака**: 55 литров
- **Расход топлива**: 10 км/л
- **Грузоподъемность**: 450 кг
- **Скорость**: Средняя (120 км/ч)
- **Прочность**: Высокая (1.2x)

## Требования к спрайтам / Sprite Requirements

### Вид сбоку (Side View)
- **Размер**: 120x80 пикселей
- **Формат**: PNG с прозрачностью
- **Цвета**: Классический бежевый/серый, можно вариации
- **Детали**: Характерная решетка радиатора, хромированные бамперы
- **Состояния**: 
  - Идеальное состояние (100%)
  - Потрепанное (50%)
  - Разбитое (0%)

### Вид сверху (Top-Down View)
- **Размер**: 64x128 пикселей
- **Формат**: PNG с прозрачностью
- **Угол**: Строго сверху для игрового вида
- **Те же состояния** что и для бокового вида

### Иконка в меню (Menu Icon)
- **Размер**: 48x48 пикселей
- **Стиль**: Упрощенный силуэт
- **Формат**: PNG с прозрачностью
- **Цвет**: Монохромный или 2-3 цвета

## Исторический контекст / Historical Context
ГАЗ-24 "Волга" производилась с 1970 по 1985 год. Была мечтой советского водителя - просторная, комфортабельная, надежная. В 90-е годы многие "Волги" использовались как такси и для дальних поездок.

## Атмосфера для арта / Art Direction
- **Эпоха**: СССР 1970-80х, эксплуатация в 90-е
- **Состояние**: Потертая, но живучая
- **Детали эпохи**: Возможны наклейки 90-х, "шашечки" такси
- **Цветовая гамма**: Приглушенные цвета, следы времени

## Примечания для художника / Artist Notes
- Сохранить узнаваемый силуэт "Волги"
- Показать характерные детали (решетка, фары, форма)
- Для пиксель-арта: сосредоточиться на характерных чертах
- Можно добавить детали эпохи (брызговики, антенна и т.д.)
