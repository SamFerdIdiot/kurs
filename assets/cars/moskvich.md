# Москвич (Moskvich) - Asset Requirements / Требования к ассетам

## Описание / Description
Компактный городской автомобиль. Москвич-412 - легенда советского автопрома. Маневренный, экономичный, с характерным дизайном. Машина для тех, кто ценит практичность.

## Технические характеристики / Technical Specifications
- **Тип**: Компактный седан / Compact City Car
- **Объем бака**: 46 литров
- **Расход топлива**: 11 км/л
- **Грузоподъемность**: 350 кг (самая малая)
- **Скорость**: Выше средней (130 км/ч)
- **Прочность**: Ниже средней (0.9x)

## Требования к спрайтам / Sprite Requirements

### Вид сбоку (Side View)
- **Размер**: 95x68 пикселей (самый компактный)
- **Формат**: PNG с прозрачностью
- **Цвета**: Зеленый, бежевый, серый - типичные цвета Москвича
- **Детали**: Характерная "горбатая" форма, округлые фары
- **Состояния**: 
  - Идеальное состояние (100%)
  - Потрепанное (50%)
  - Разбитое (0%)

### Вид сверху (Top-Down View)
- **Размер**: 52x104 пикселей (самый маленький)
- **Формат**: PNG с прозрачностью
- **Угол**: Строго сверху для игрового вида
- **Те же состояния** что и для бокового вида

### Иконка в меню (Menu Icon)
- **Размер**: 48x48 пикселей
- **Стиль**: Упрощенный силуэт Москвича
- **Формат**: PNG с прозрачностью
- **Цвет**: Монохромный или 2-3 цвета

## Исторический контекст / Historical Context
Москвич-412 производился с 1967 по 1975 год, но эксплуатировался до 90-х годов. Компактный городской автомобиль с характерной формой. В 90-е часто использовался как городская машина для небогатых водителей. Славился ремонтопригодностью и простотой.

## Атмосфера для арта / Art Direction
- **Эпоха**: 1970-е производство, эксплуатация в 90-е
- **Состояние**: Часто в плохом состоянии, но на ходу
- **Детали эпохи**: Следы многократного ремонта, разномастные детали
- **Цветовая гамма**: Приглушенные цвета, выцветшая краска

## Примечания для художника / Artist Notes
- Сохранить характерную компактную форму Москвича
- Округлые фары - важная черта
- Более компактный чем другие машины
- Для пиксель-арта: подчеркнуть округлые формы
- Показать простоту и утилитарность

## Особенности / Special Features
- Самая компактная машина из трех
- Маневренная и экономичная
- Характерный "народный" дизайн
- Подходит для городской среды

## Игровая роль / Gameplay Role
- Для игроков, выбирающих экономию и маневренность
- Меньше места для груза, но легче управлять
- Хорошо для городских локаций и коротких поездок
