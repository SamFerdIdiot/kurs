**Regions:**

### Interactive Elements

**Nodes (Location Icons):**
- **Type:** Clickable Circles
- **Size:** 40x40 pixels
- **States:**
  - Unvisited: Gray circle with icon
  - Current: Highlighted with pin icon
  - Visited: Colored circle with checkmark
  - Hovered: Glow effect + tooltip
  - Connected: Can be clicked
  - Disconnected: Grayed out, unclickable

**Routes (Connecting Lines):**
- **Type:** Lines between nodes
- **Width:** 4 pixels
- **States:**
  - Available: White solid line
  - Traveled: Red solid line
  - Locked: Gray dashed line

## Cinematic Travel Scene

### Overview
Immersive travel experience between locations with parallax scrolling and event system.

### Layout Specifications

**Screen Resolution:** 1366x768  
**Layout Type:** Layered rendering

**Layers:**
1. Background Sky (Layer 0) - Slowest parallax
2. Far Mountains (Layer 1) - Slow parallax
3. Trees/Buildings (Layer 2) - Medium parallax
4. Road Markers (Layer 3) - Fast parallax
5. Vehicle (Fixed center-left)
6. HUD Overlay (Layer 6)
7. Event Overlay (Layer 7)
