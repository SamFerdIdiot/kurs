## –û–±–∑–æ—Ä —Å–∏—Å—Ç–µ–º—ã / System Overview

### English

The **Ability Tree System** is a character progression mechanic inspired by "Keep Driving" that allows players to unlock and upgrade both **passive abilities** (always active bonuses) and **active perks** (usable skills during events). The system provides meaningful choices that affect gameplay during road events, resource management, and character interactions.

**Key Features:**
- **5 ability categories** with distinct progression paths
- **Active perks** usable during events (e.g., "Relax", "Negotiate", "Repair")
- **Passive bonuses** that continuously improve character performance
- **Level-gated progression** requiring experience points
- **Cooldown system** to balance active perk usage
- **Resource costs** for powerful abilities
